کوالێتی، ئاسایش و پشتگیری

ئەم پەڕەیە چوارچێوەی فەرمی AI Gate Iraq بۆ پاراستنی کارایی، ئاسایش، بەڵگەنامە، فێرکردنی بەکارهێنەر و پشتگیری ڕوون دەکاتەوە.

پڕۆتۆکۆلی تاقیکردنەوەی کارایی

هەر گۆڕانکارییەکی production پێویستە پشکنینی TypeScript، build و project guard تێپەڕێنێت. هەروەها تاقیکردنەوەی خوێندنەوەی HTTP بۆ website و platform بە شێوەیەکی کۆنترۆڵکراو ئەنجام دەدرێت.

ئامانجە سەرەکییەکان: LCP لە ٢.٥ چرکە کەمتر، CLS لە ٠.١٠ کەمتر، p95 ـی داواکارییە static ـەکان لە ١.٥ چرکە کەمتر و ڕێژەی هەڵە لە ١٪ کەمتر.

تاقیکردنەوەی بار و stress تەنها لە Firebase Emulator یان ژینگەی non-production ئەنجام دەدرێت؛ هیچ داتای ساختە لە production Firestore دروست ناکرێت.

پڕۆتۆکۆلی پشکنینی ئاسایش

پشکنینەکان Firebase Authentication، Firestore rules، admin authorization، زانیاریی نهێنی، dependency ـەکان، XSS، injection، broken access control و security headers دەگرنەوە.

کێشەی critical یان high وەک دەستگەیشتنی بێ‌مۆڵەت، خوێندنەوەی public ـی داتای تایبەت، stored XSS یان ئاشکرابوونی credential مانعی release و deploy دەبێت تا چاکسازی و retest تەواو بێت.

ڕاپۆرتی ئاسایش بە نهێنی بۆ support@idg.official.iq بنێرە و هیچ token، password یان داتای ڕاستەقینەی بازرگان تێدا مەخە.

بەڵگەنامەی تەکنیکی

پلاتفۆرم React، TypeScript، Vite و Firebase بەکاردەهێنێت. Website و Platform لە Firebase Hosting ـن؛ Firebase Authentication ناسنامە، Firestore داتا و Cloudflare DNS/CDN/Security بەڕێوەدەبات.

لە ئێستادا public partner API بەردەست نییە. هەر integration ـێکی داهاتوو پێویستی بە contract ـی versioned، authentication، authorization، quota، audit logging و پشکنینی ئاسایش هەیە.

بۆ داواکاریی integration یان بەڵگەنامەی زیاتر پەیوەندی بە تیمی پشتگیری بکە.

ڕێنمایی بەکارهێنەر

  1. پلاتفۆرم بکەرەوە و زمانی کوردی یان عەرەبی هەڵبژێرە.
  2. لە سایدبارەوە «چوونەژوورەوە» هەڵبژێرە و بە هەژماری Google ـی ڕێگەپێدراو بچۆ ژوورەوە.
  3. خزمەتگوزاری گونجاو وەک ڕاوێژکاری زیرەک، دراو، تێچوو، KYC، دابینکردن یان بەدواداچوونی بار هەڵبژێرە.
  4. زانیاریی پێویست بە وردی بنووسە و پێش بڕیاری یاسایی، دارایی یان گومرگی لە سەرچاوەی فەرمی پشتڕاستی بکەوە.
  5. لە ئامێری هاوبەشدا دوای تەواوبوون دەرچوون بکە و password، token یان زانیاریی پارەدان لە chat مەنووسە.

تیمی پشتگیری

ئیمەیڵی فەرمی پشتگیری: support@idg.official.iq

لە نامەکەتدا ناو و کۆمپانیا، URL ـی کێشەکە، بەروار و کات، browser و ئامێر، هەنگاوەکانی دووبارەکردنەوە و screenshot ـێک بەبێ زانیاریی حەساس بنووسە.

کێشەی production، ئاشکرابوونی داتا یان دەستگەیشتنی بێ‌مۆڵەت بە پلەی بەرز مامەڵەی لەگەڵ دەکرێت و بۆ خاوەنی پلاتفۆرم escalation دەکرێت.
